# 🎬 Movie CRUD Web App
A simple and interactive Movie Management Web Application that allows users to perform full CRUD operations on movies.

## 🚀 Live Demo
🔗 https://mahmoud-657.github.io/Movie-CRUD-Web-App/


## 📌 Features
- ➕ Add new movies with details
- 📄 View all movies in a clean UI
- ✏️ Update/edit movie information
- ❌ Delete movies 
- 💾 Data stored locally (Local Storage)
- 📱 Responsive design (works on all devices)


## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript   
- Bootstrap  


## ⚙️ How It Works
This project is based on the concept of **CRUD Operations**:
- **Create** → Add a new movie  
- **Read** → Display movies  
- **Update** → Edit movie details  
- **Delete** → Remove a movie
